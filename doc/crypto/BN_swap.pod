=head1 NAME

BN_swap - exchange BIGNUMs

=head1 SYNOPSIS

 #include <openssl/bn.h>

 void BN_swap(BIGNUM *a, BIGNUM *b);

=head1 DESCRIPTION

BN_swap() exchanges the values of I<a> and I<b>.
